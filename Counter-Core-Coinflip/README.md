# Counter Core Coinflip Addon

In this repository resides the core script coinflip addon.

## Getting Started

You will need to deploy the main system which can be found [Counter Core](https://github.com/ColinMoulds/Counter-Core).

### Deployment

Drag & Drop the web files in /var/www/html.

Upload the .sql database through your PHPMyadmin.
